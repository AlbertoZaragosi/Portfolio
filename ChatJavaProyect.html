Chat Server Client

This Java code represents a basic implementation of client-server communication using sockets. The application consists of two components: a Server and a Client. The server waits for a client to connect and then reads and prints messages received from the connected client. The client connects to the server, sends messages, and can terminate the connection by typing "bye."
